Project 1

"Deep Learning System for Fire Detection and Alerts using YOLO"
This project uses the YOLO (You Only Look Once) deep learning model for real-time fire detection. When fire is detected, the system draws bounding boxes around the fire and displays confidence scores. Simultaneously, it triggers an alarm, sends an email with the fire image to authorities, and activates water pumps via an ESP32 microcontroller. This ensures a swift response to fire incidents, enhancing safety and minimizing damage.
